Clôture de période & Export CSV
Cette opération exporte l'intégralité de toutes les feuilles HACCP en un fichier CSV horodaté, puis efface toutes les données pour repartir sur une période vierge. La traçabilité est conservée via le fichier CSV téléchargé.
  1. Toutes les feuilles (Réception, Froid, Huiles, Refroidissement, Congélation, Remise en T°,Historique de nettoyage, log et actionlog) sont exportées en CSV
  2. Le fichier CSV est téléchargé sur votre appareil avant toute suppression
  3. Un log de clôture est enregistré dans le journal des connexions
  4. Toutes les lignes de données sont effacées de la base
